## Project Overview

Reverse-engineering specification and tooling for the Windows 11 Start Menu persistence file (`start2.bin`). The file is encrypted with AES-256-CBC using keys derived from two MT19937 PRNGs seeded from the file's own FILETIME header field — properly described as obfuscation, not authenticated encryption.

## Repository Structure

- `README.md` / `README.zh_TW.md` — Authoritative format specification (wire format, crypto constants, MT19937 parameters, worked example, reference Python implementation)
- `StartMenuBin/StartMenuBin.psm1` — PowerShell 5.1 module implementing the spec. Exports `Unprotect-StartMenuBin` (decrypt) and `Protect-StartMenuBin` (encrypt). Handles registry backup to `RoamedTilePropertiesMap` cache key.
- `references/` — Windows 11 v2126.5401.10.0 binaries (`StartMenu.dll`, `StartMenuExperienceHost.exe`, `SHCore.dll`), sample `start2.bin`, its decrypted JSON (`start2.json`), and a kernel call-stack trace (`caller.CSV`) showing the file write path.

## Key Technical Details

- **Crypto**: AES-256-CBC + PKCS#7. Key is first 32 bytes of a 64–128 byte alphanumeric string; IV is 16 bytes. Both generated by `AlphaNumericKeyGenerator` backed by MT19937.
- **MT19937**: Standard C++11 `std::mt19937` (state size 624, multiplier `0x6C078965`, matrix `0x9908B0DF`). Two independent instances per file: `pad_mt` (padding split + random fill) and `sym_mt` (key/IV generation).
- **Frame layout**: 16B MAGIC + 16B HEADER_CONST + 8B FILETIME + 4B total_payload_length + variable pre-padding + ciphertext + variable post-padding. Total padding always 512 bytes.
- **Registry**: Encrypted blob is also written to `HKCU\Software\Microsoft\Windows\CurrentVersion\CloudStore\Store\Cache\DefaultAccount\$...roamedtilepropertiesmap`
- **PowerShell module** uses `System.Security.Cryptography.Aes` (managed) for AES and `[BitConverter]` for LE parsing. Compatible with PowerShell 5.1+ (Windows built-in).

## PowerShell Module Usage

```powershell
Import-Module .\StartMenuBin\StartMenuBin.psm1

# Decrypt the live start menu file
Unprotect-StartMenuBin

# Decrypt a specific file
Unprotect-StartMenuBin -Path "C:\path\to\start2.bin"

# Encrypt and write back (also updates registry by default)
Protect-StartMenuBin -PlainText $jsonString
```
